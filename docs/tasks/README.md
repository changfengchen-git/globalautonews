# GlobalAutoNews — 开发任务总览

> 本目录包含项目的全部开发任务，按 Phase 和编号组织。
> 每个任务文件可以独立交给 AI Coding 工具执行。

## 任务执行原则

1. **严格按编号顺序执行**，每个任务标注了前置依赖
2. **每个任务完成后**，执行其中的"验收标准"确认通过
3. **不要跨任务修改文件**，除非明确标注
4. **所有代码写在 `/Users/apple/Desktop/globalautonews/` 目录下**
5. **Python 版本 3.11+**，使用 type hints
6. **所有 Python 文件使用 UTF-8 编码**
